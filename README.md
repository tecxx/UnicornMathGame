A fun little math learning game for kids. Created while doing homework with my young ones.
Written in c# .net core 10 / compile with visual studio 


Goal: Solve random math questions to advance your unicorn to the castle and reveal puzzle tiles.

<img width="1326" height="893" alt="image" src="https://github.com/user-attachments/assets/105474c3-3573-4002-ab07-77c8968567dc" />


Features four game modes (addition, substraction, multiplication and vision) in three difficulty levels. Scoreboard included.
